## 📖 About the Project

**Profile Card Generator** is a simple yet feature-rich web application built using **HTML, CSS, and JavaScript**. The project allows users to generate and manage profile cards dynamically while providing hands-on experience with core JavaScript and DOM concepts.

This project was developed as part of my **Full Stack / MERN Stack Development learning journey**, where I focused on understanding how JavaScript interacts with HTML elements and how dynamic user interfaces are created.

---

## ✨ Features

* Create dynamic profile cards using form inputs
* Real-time DOM manipulation
* Interactive user interface
* Responsive card layout
* Smooth hover effects and animations
* Clean and modern design

---

## 🛠️ Technologies Used

* **HTML5**
* **CSS3**
* **JavaScript (Vanilla JS)**
